## How to view it for yourself
1. Copy or download the [Diagrams](/Diagrams.txt) file.
2. Go to [PlantText](https://www.planttext.com/).
3. Open or paste the [Diagrams](/Diagrams.txt) filecontent in the editor.
4. Click on the ``Refresh`` button or press the ``Alt + Enter`` keys on your keyboard.
5. Finally, you should see the current version of the [UML Diagrams](/Diagrams.txt) as depicted above.
